# GitHub Pages 써먹기

개인 웹사이트를 만들어보고 싶단 생각은 한참 전부터 했다. 도메인을 산지 약 5개월만에 실제로 뭔가를 만들었다.

## 여기까지 오기 위해 대충 필요한 준비물

### 도메인

도메인 생각보다 얼마 안한다. 1년에 15$면 원하는 주소 구할 수 있다.

### Cloudflare 계정

DNS 관리하기 편해서 그냥 계정 만들었다. 무료이고 필수는 아니다.

### AWS 인스턴스

아직은 안쓰고 있지만 나중에 직접 서버 돌리는게 편하단 생각 들면 써먹으려고 만들어놓기만 했다.

## 앞으로 할 일

### Jekyll 좀 더 뜯어보기

사이트가 잘 동작하는지 보려고 매번 master에 커밋할수는 없잖아. 로컬에 환경 준비해놓고 쓰고 있는데 이거 틀어놓으면 실시간으로 파일 변경사항 반영해서 웹에 올려주는 것 같다.
